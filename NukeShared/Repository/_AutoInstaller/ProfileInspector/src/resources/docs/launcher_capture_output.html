Launch Nuke Instance

From this section you can launch a new Nuke instance with the profiler listener active, so to create the xml report file.

By default the Nuke instance will be created with the current Nuke executable. However, it can be changed to a different one.
Also, when selecting the nuke executable, be sure to select the base Nuke and not the other modes.

You can also specify the mode in which Nuke's will launch (NukeX, NukeStudio, etc.).

Other options

  • Capture output
  • Dock undock window
  • Set Optional arguments (Refer to Nuke Command Line Operations for more information)

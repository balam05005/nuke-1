DAG Table

NOTE: When opening a new Nuke session, the table will be empty. To update its content you must click on the Refresh Table button.

The table shows the current nodes present in the DAG and their respective profiling stats if profiling listener is activated.

Order/Sort

The table allows to sort columns in ascending or descending order by clicking on a column header.

Navigation/Interactions

The table can be navigated by mouse or arrow keyboard when the table is in focus.

  • Up/Down arrows will move the selection and zoom on the current selected node.
  • Left-click on a node name, it will automatically zoom the node inside the dag.
  • Left double-click on a node name, it will open its proprieties panel.

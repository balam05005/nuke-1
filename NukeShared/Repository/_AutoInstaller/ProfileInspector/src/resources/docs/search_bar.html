Search Bar

The search bar allows you to filter the nodes present in the Node Graph, with real time feedback of the result.
It also accepts regular expression patterns.

Options

Case Sensitive:
Toggle case sensitive.
Whole Word:
Search by whole word (equivalent to regex: ^text).
Filter by:
  • Name - name of the Node.
  • Type - type (class) of the node.
  • Mix - Both name and type. (eg: by writing r it will find both names and classes that have an r).

    • Advanced Options

    NOTE: This section will not be present in the XML Report page.

    By toggling the arrow on the left side of the search bar, some other options will be shown. Those options state will be saved between sessions.

    Zoom when search:
    If enabled, when performing a search, it will automatically zoom on the first node that matches the search pattern.
    Force floating properties:
    When double-clicking on a node name, will load its proprieties on a floating window rather than inside Proprieties panel.
    Exclude nodes:
    Node Types (class) to be hidden in the table view. The names can be written with a title case or lower case and delimited by everything e.g. dot Viewer, camera.
